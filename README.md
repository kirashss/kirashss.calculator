# Калькулятор

Приложение калькулятор на JavaScript.

## О проекте
Простое приложение, используемое для выполнения арифметические вычислений.

## Функциональнсть
```
- сложение
- вычитание
- умножение
- деление
- сброс операции
```

## Используемые технологии

* CSS
* HTML
* JavaScript

## Дальнейшие доработки
* Работа с процентами
* Переключение числа на отрицательное и обратно
